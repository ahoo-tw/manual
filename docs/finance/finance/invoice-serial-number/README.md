
## 步驟一：選擇發票類型
### 『二聯式』－應稅
開立國內銷貨發票給一般消費者。  
無記載買受人名稱及統一編號。  
### 『三聯式』－應稅
開立國內銷貨發票給營業人。  
### 『核銷』
政府補助企劃以核銷方式作為收款依據。  
### 『INVOICE』
產品屬零稅率銷售。

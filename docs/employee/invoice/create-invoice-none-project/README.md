## 步驟一：選擇發票類型

![非專案發票](./unproject.gif)
| 類型 | 稅率 | 說明 |
| ------- | ------ | -------------------------------------------------------------- |
| 二聯式 | 應稅 | 開立國內銷貨發票給一般消費者。<br>無記載買受人名稱及統一編號。 |
| 三聯式 | 應稅 | 開立國內銷貨發票給營業人。 |
| INVOICE | 零稅率 | 產品屬零稅率銷售。 |
